# Geo-Analisis

## 01 | OpenStreetMap

[OpenStreetMap](https://www.openstreetmap.org/) es un sitio para cartografiar objetos reales y actuales.  
Esto incluye millones de edificios, carreteras y otros datos de destinos.  
Es posible cartografiar cualquier rasgo del mundo real que resulte de interés.

A partir de los datos del proyecto **OpenStreetMap** se pueden producir:

- Mapas de carreteras  
- Mapas de senderismo  
- Mapas de vías ciclables  
- Mapas náuticos  
- Mapas de estaciones de esquí  

También se utilizan en aplicaciones para el cálculo de rutas óptimas tanto para vehículos como para peatones.

### Repositorio oficial

- GitHub: https://github.com/openstreetmap/


## 02 | Leaflet

[Leaflet](https://leafletjs.com/) es la principal biblioteca JavaScript de código abierto para la creación de mapas interactivos compatibles con dispositivos móviles.

Es una biblioteca líder en mapas interactivos para móviles. Con un peso de tan solo **42 KB de JavaScript**, ofrece todas las funcionalidades de mapeo que la mayoría de los desarrolladores necesitan.

Leaflet ha sido diseñada pensando en la **sencillez**, el **rendimiento** y la **facilidad de uso**.  
Funciona de manera eficiente en las principales plataformas de escritorio y móviles, y puede ampliarse mediante una gran variedad de plugins.

### Recursos

- Repositorio GitHub: https://github.com/Leaflet/Leaflet  
- Descarga oficial: https://leafletjs.com/download.html

## 03 | Mapbox

[Mapbox](https://www.mapbox.com/) proporciona potentes motores de enrutamiento, tiempos de viaje precisos basados en el tráfico e intuitivas indicaciones giro a giro para ayudar a crear atractivas experiencias de navegación.

Ofrece mapas interactivos y totalmente personalizables en el navegador, utilizando tecnología de **mosaicos vectoriales** y **WebGL**.

Mapbox pone a disposición una amplia variedad de **herramientas y APIs** que permiten a los desarrolladores incorporar mapas interactivos y capacidades de ubicación en aplicaciones y sitios web.

### Recursos

- Repositorio GitHub: https://github.com/mapbox

## 04 | QGIS

[QGIS](https://www.qgis.org/es/site/) es un sistema de información geográfica (**SIG**) de código abierto y gratuito que permite la **visualización**, **análisis** y **edición** de datos geoespaciales.

Es una herramienta ampliamente utilizada por profesionales y entusiastas de los SIG en diversas disciplinas, como:

- Cartografía  
- Planificación urbana  
- Gestión ambiental  
- Agricultura  
- Geomática  
- Entre otras

QGIS ofrece múltiples funcionalidades, entre ellas:

- Visualización de mapas  
- Análisis espacial  
- Edición de datos  
- Composición de mapas  
- Integración de datos externos  
- Plugins y personalización  

### Recursos

- Repositorio GitHub: https://github.com/mapbox  
- Descarga oficial: https://www.qgis.org/es/site/forusers/download.html


## 05 | GeoServer

[GeoServer](https://geoserver.org/) es un servidor de datos espaciales de **código abierto**, escrito en **Java**, que permite a los usuarios **compartir y editar datos geoespaciales**.

Diseñado para la **interoperabilidad**, GeoServer publica datos procedentes de cualquier fuente de información espacial utilizando **estándares abiertos**.

GeoServer destaca por:

- Una comunidad activa y actualizaciones frecuentes  
- Soporte para una amplia variedad de orígenes de datos  
- Una interfaz amigable y fácil de usar  

### Recursos

- Repositorio GitHub: https://github.com/geoserver/geoserver  
- Descarga oficial: https://geoserver.org/download/


## 06 | GRASS GIS

[GRASS GIS](https://grass.osgeo.org/) es un sistema de información geográfica de **código abierto** que, mediante su **plugin en QGIS**, proporciona acceso a las bases de datos y a las funcionalidades SIG de GRASS.

El plugin de GRASS en QGIS incluye, entre otras, las siguientes capacidades:

- Herramientas de geoprocesamiento para datos vectoriales  
- Conversión de formatos (ráster ↔ vectorial)  
- Herramientas de análisis y extracción para datos ráster  
- Procesamiento de imágenes de satélite  
- Herramientas de corrección topológica  
- Análisis de redes de transporte  
- Cálculo de cuencas hidrográficas  
- Transformación de medidas de rumbo y distancia  

### Recursos

- Repositorio GitHub: https://github.com/geoserver/geoserver


## 07 | MapServer

[MapServer](https://mapserver.org/) es una plataforma de **código abierto** para la publicación de **datos espaciales** y el desarrollo de **aplicaciones de mapas interactivos en la web**.

MapServer destaca por las siguientes características:

- Multiplataforma  
- Alta velocidad de acceso a los datos  
- Soporte para una amplia variedad de formatos ráster y vectoriales  
- Integración con numerosos productos que lo soportan, como clientes web (ka-Map, Chameleon, CartoWeb) y utilidades para el tratamiento de imágenes  

### Recursos

- Repositorio GitHub: https://github.com/MapServer/MapServer/


## 08 | OpenLayers

[OpenLayers](https://openlayers.org/) facilita la incorporación de **mapas dinámicos** en cualquier página web.

Permite mostrar **mosaicos de mapas**, **datos vectoriales** y **marcadores** cargados desde cualquier fuente.  
OpenLayers ha sido desarrollado para promover el uso de información geográfica de todo tipo.

Es una biblioteca **JavaScript de código abierto**, completamente gratuita, publicada bajo la **licencia BSD de 2 cláusulas** (también conocida como FreeBSD).

### Recursos

- Repositorio GitHub: https://github.com/openlayers/openlayers  
- Descarga oficial: https://openlayers.org/download/


## 09 | PostGIS

[PostGIS](https://postgis.net/) es la **extensión espacial para PostgreSQL** que permite almacenar, consultar y trabajar con **datos geométricos y geoespaciales**.  
Es una herramienta impulsada por **OSGeo** y resulta indispensable en proyectos **GIS** basados en bases de datos relacionales.

PostGIS amplía las capacidades de PostgreSQL, dotándolo de múltiples ventajas para el manejo de información espacial, como:

- Almacenamiento de datos geométricos  
- Consultas espaciales avanzadas  
- Soporte para estándares geoespaciales  
- Integración con herramientas y aplicaciones GIS  

### Recursos

- Repositorio Git: https://git.osgeo.org/gitea/postgis/postgis


## 10 | Cesium

[Cesium](https://cesium.com/) es una plataforma y conjunto de herramientas para la **visualización y análisis de datos geoespaciales en 3D**.  
Es una tecnología basada en web que permite la representación de datos geoespaciales en **tiempo real** y de manera **interactiva**.

Con Cesium 3D Geodata es posible visualizar y explorar información geográfica en un entorno tridimensional altamente realista, incluyendo:

- Terrenos  
- Edificios  
- Árboles  
- Cuerpos de agua  
- Otros elementos geográficos  

### Recursos

- Repositorio GitHub: https://github.com/CesiumGS


## 11 | Mapnik

[Mapnik](https://mapnik.org/) es un conjunto de **herramientas para la renderización de mapas** que ofrece:

- Bordes limpios y características suaves gracias a gráficos de calidad anti-distorsión  
- Colocación inteligente de etiquetas  
- Simbolización escalable en formato SVG  

Mapnik es ampliamente conocido por ser utilizado para **representar las capas principales en OpenStreetMap**.

Generalmente, Mapnik se integra en aplicaciones **Python** para entregar mapas a través de Internet.  
Recientemente, también se ha usado para **crear mapas de alta resolución para impresión** debido a sus mejoras de escalabilidad.

### Recursos

- Repositorio GitHub: https://github.com/mapnik/mapnik


## 12 | GDAL

[GDAL](https://gdal.org/) es una **biblioteca traductora de formatos de datos geoespaciales** tanto **ráster** como **vectoriales**, publicada por la **Open Source Geospatial Foundation** bajo una **licencia de código abierto tipo MIT**.

Como biblioteca, GDAL ofrece:

- Un **modelo abstracto único** para datos ráster  
- Un **modelo abstracto único** para datos vectoriales  
- Compatibilidad con **múltiples formatos geoespaciales**  
- Una serie de **utilidades de línea de comandos** para traducción y procesamiento de datos  

### Recursos

- Descarga oficial: https://gdal.org/download.html

## 13 | Mapzen

[Mapzen](https://www.mapzen.com/) es una plataforma de **datos geoespaciales de código abierto** que proporciona herramientas para trabajar con datos ráster y vectoriales.  
Está diseñada para facilitar la integración de mapas y servicios geoespaciales en aplicaciones web y móviles.

Mapzen ofrece:

- Un **modelo abstracto único** para datos ráster  
- Un **modelo abstracto único** para datos vectoriales  
- Utilidades y APIs para traducción y procesamiento de datos geoespaciales  

### Recursos

- Repositorio GitHub: https://github.com/mapzen/


## 14 | Kepler.gl

[Kepler.gl](https://kepler.gl/) es una plataforma de **visualización de datos geoespaciales de código abierto** desarrollada por **Uber**.  
Ofrece una **interfaz interactiva y poderosa** para explorar y visualizar datos en un entorno cartográfico.

Kepler.gl se destaca por:

- Visualización de **grandes volúmenes de datos geoespaciales en tiempo real**  
- Posibilidad de **cargar y combinar múltiples capas** de datos (puntos, líneas, polígonos)  
- Amplia gama de opciones de **visualización y análisis**  

### Recursos

- Repositorio GitHub: https://github.com/keplergl/kepler.gl


## 15 | D3.js

[D3.js](https://d3js.org/) es una **biblioteca JavaScript de código abierto** para crear **visualizaciones interactivas y personalizadas**, incluyendo visualización de datos geoespaciales.

Permite:

- Crear mapas combinando datos geoespaciales con **SVG** (Scalable Vector Graphics)  
- Representar formas geográficas como **polígonos, líneas y puntos**  
- Manipular y visualizar datos geoespaciales de manera **flexible y personalizada**  

### Recursos

- Repositorio GitHub: https://github.com/d3


## 16 | Marble

[Marble](https://marble.kde.org/) es un programa de **código abierto** del Proyecto de Educación KDE, similar a **NASA World Wind** o **Google Earth**.

Marble permite:

- Elegir entre múltiples mapas, incluyendo **OpenStreetMap**  
- Integrar un **widget de Marble** en aplicaciones propias  
- Explorar mapas de ciudades y calles, buscar direcciones y lugares de interés  
- Consultar varios backends de búsqueda y presentar resultados en una **vista unificada**  
- Calcular rutas para **peatones, bicicletas y automóviles**, tanto en línea como fuera de línea, con múltiples puntos intermedios  

### Recursos

- Descarga oficial: https://marble.kde.org/install.php


## 17 | Folium

[Folium](http://python-visualization.github.io/folium/) es una **biblioteca de Python** que se basa en **Leaflet.js** para crear **mapas interactivos** en el navegador web.

Permite:

- Agregar **capas, marcadores y controles interactivos** a los mapas  
- Visualizar datos **geoespaciales** de manera interactiva  
- Integrarse con **Python y Jupyter Notebook** para análisis y visualización de datos  

### Recursos

- Descarga oficial: http://python-visualization.github.io/folium/



## 18 | Orfeo

[Orfeo](https://www.orfeo-toolbox.org/) es un proyecto de **código abierto** para **detección remota de última generación**.

Permite:

- Procesar imágenes **ópticas, multiespectrales y de radar**  
- Manejar datos de **alta resolución** a escala de **terabytes**  
- Aprovechar la experiencia de la **comunidad geoespacial de código abierto**  

### Recursos

- Descarga oficial: https://www.orfeo-toolbox.org/download


## 19 | PySAL

[PySAL](https://pysal.org) es una **librería de Python** que permite realizar distintos tipos de **análisis espacial** y ciencia de datos espaciales.

PySAL incluye cuatro módulos principales que forman la base de muchos paquetes de la familia PySAL:

- **Pesos espaciales:** `libpysal.weights`  
- **Entrada y salida de datos:** `libpysal.io`  
- **Geometría computacional:** `libpysal.cg`  
- **Conjuntos de datos de ejemplo integrados:** `libpysal.examples`  

### Recursos

- Descarga oficial: https://pysal.org/docs/install/


## 20 | Whitebox Geo

[Whitebox Geo](https://www.whiteboxgeo.com/) es una herramienta para **detección remota y procesamiento de imágenes**, que permite:

- Mejora de imágenes y creación de mosaicos  
- Operaciones de filtrado  
- Clasificación simple (k-means)  
- Transformaciones comunes de imágenes  

También permite realizar operaciones de **análisis SIG** como:

- Análisis de costo-distancia  
- Almacenamiento en búfer de distancia  
- Reclasificación de ráster  
- Y muchas otras tareas geoespaciales  

### Recursos

- Descarga / Repositorio GitHub: https://github.com/jblindsay/whitebox-too


## 21 | Mapillary

[Mapillary](https://www.mapillary.com/) es un **servicio de street view por crowdsourcing**.  
Fundada en 2013 en Malmö, Suecia, por Jan Erik Solem y Johan Gyllenspetz, fue adquirida por **Meta** en 2020.

Permite:

- Acceder a imágenes a nivel de calle y datos cartográficos de todo el mundo  
- Completar espacios vacíos creando tus propias fotos o vídeos  

### Recursos

- Repositorio GitHub: https://github.com/mapillary


## 22 | GeoTools

[GeoTools](https://www.geotools.org/) es una **biblioteca Java de código abierto (LGPL)** que proporciona métodos conformes a los estándares para la **manipulación de datos geoespaciales**, útil para implementar **Sistemas de Información Geográfica (SIG)**.

GeoTools implementa las **especificaciones del Open Geospatial Consortium (OGC)** a medida que se desarrollan, asegurando compatibilidad con estándares geoespaciales.

### Recursos

- Descarga oficial: https://sourceforge.net/projects/geotools/files/
